    0.虚拟机安装VMware Tools工具

    打开Vmware，选择菜单栏中的“虚拟机”–>安装Vmware Tools;

    在linux的文件管理器里打开光驱的界面，打开终端，执行sudo mv VMwareTools-10.2.5-8068393.tar.gz /opt/；

    cd /opt 来到opt目录下；

    tar zxpf VMwareTools-10.2.5-8068393.tar.gz 解压；

    cd vmware-tools-distrib 进入解压后的vmware-tools-distrib目录；

    sudo ./vmware-install.pl；

    输入密码，一路Enter即可。

    注：安装完成后即可全屏显示，但有些功能需重启生效（比如与win10之间的Ctrl c/v🤣)

    参考：https://zhuanlan.zhihu.com/p/67622390
